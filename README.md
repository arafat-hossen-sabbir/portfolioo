# Arafat hossen sabbir Portfolio

A modern, responsive portfolio website built with Next.js 14, Tailwind CSS, and custom scroll animations.

## Features

- 🌙 Dark/Light mode toggle (persisted in localStorage)
- ✨ Smooth scroll animations (intersection observer)
- 📱 Fully responsive design
- 🎨 Dynamic skill progress bars (animate on scroll)
- 🗂️ Filterable project cards
- 📬 Contact form
- 🔗 Project detail pages

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Customization

Edit `src/app/data.js` to update:
- Personal info, social links, contact details
- Skills and proficiency levels
- Education and experience
- Projects

## Deployment

Deploy instantly to Vercel:

```bash
npm run build
```

Or push to GitHub and connect to [vercel.com](https://vercel.com).
